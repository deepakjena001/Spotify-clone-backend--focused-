# Spotify Clone – Backend
A backend-focused Spotify clone built using Node.js and Express, implementing RESTful APIs for authentication, music management, and playlist handling.

# Features
🔐 User Authentication (Signup/Login) <br>
🎵 Music data handling (songs, albums, etc.)<br>
📂 Playlist creation and management<br>
📡 RESTful API architecture<br>
🗄️ Database integration (MongoDB)<br>
⚙️ Scalable backend structure<br>

# Tech Stack
Node.js<br>
Express.js<br>
MongoDB<br>
Mongoose<br>
JWT (Authentication)<br>
REST API<br>
