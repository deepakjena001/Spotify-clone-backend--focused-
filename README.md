# Spotify Clone – Backend
A backend-focused Spotify clone built using Node.js and Express, implementing RESTful APIs for authentication, music management, and playlist handling.

# Features
🔐 User Authentication (Signup/Login)
🎵 Music data handling (songs, albums, etc.)
📂 Playlist creation and management
📡 RESTful API architecture
🗄️ Database integration (MongoDB)
⚙️ Scalable backend structure

# Tech Stack
Node.js
Express.js
MongoDB
Mongoose
JWT (Authentication)
REST API
